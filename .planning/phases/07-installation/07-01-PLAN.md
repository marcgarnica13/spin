    1. Add two new variables at the top (after `PREFIX ?= /usr/local`):
       ```

       ```

    2. Update `.PHONY` to include all four targets:
       ```
       .PHONY: install install-extension uninstall uninstall-extension
       ```

    3. Rename the existing `install` target body to a new `install-cli` target:
       ```
       install-cli:
       	@mkdir -p $(PREFIX)/bin $(PREFIX)/lib/spin
       	@cp bin/spin $(PREFIX)/bin/spin
       	@chmod +x $(PREFIX)/bin/spin
       	@cp lib/*.sh $(PREFIX)/lib/spin/
       	@echo "spin CLI installed to $(PREFIX)/bin/spin"
       ```

    4. Add a new `install-extension` target:
       ```
       install-extension:
       	@mkdir -p $(EXTENSION_DEST)
       	@cp gnome-extension/metadata.json $(EXTENSION_DEST)/
       	@cp gnome-extension/extension.js $(EXTENSION_DEST)/
       	@echo "Spin extension installed to $(EXTENSION_DEST)"
       ```

    5. Update the `install` target to depend on both:
       ```
       install: install-cli install-extension
       	@echo "Next: run 'gnome-extensions enable spin@gsd.local'"
       ```

    6. Rename the existing `uninstall` target body to `uninstall-cli`:
       ```
       uninstall-cli:
       	@rm -f $(PREFIX)/bin/spin
       	@rm -rf $(PREFIX)/lib/spin
       	@echo "spin CLI uninstalled"
       ```

    7. Add a new `uninstall-extension` target:
       ```
       uninstall-extension:
       	@rm -rf $(EXTENSION_DEST)
       	@echo "Spin extension uninstalled"
       ```

    8. Update the `uninstall` target to depend on both:
       ```
       uninstall: uninstall-cli uninstall-extension
       	@echo "Spin uninstalled"
       ```

    IMPORTANT: Makefile uses TAB characters for recipe indentation (not spaces). Ensure all recipe lines use a literal tab character.

    Final Makefile structure (top to bottom):
    - Variables block: PREFIX, EXTENSION_UUID, EXTENSION_DEST

    - install target (depends on install-cli install-extension)

    - install-extension target (new extension copy logic)
    - uninstall target (depends on uninstall-cli uninstall-extension)
